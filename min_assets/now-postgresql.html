PostgreSQL функция now

В этом учебном пособии вы узнаете, как использовать PostgreSQL функцию now с синтаксисом и примерами.

Описание

Функция now PostgreSQL возвращает текущую дату и время с часовым поясом.

Синтаксис

Синтаксис функции now в PostgreSQL:

now( )

Параметры или аргументы

Для функции now нет параметров или аргументов.

Примечание

  • Функция now вернет текущую дату в формате 'YYYY-MM-DD HH:MM:SS.US+TZ'.

Применение

Функцию now можно использовать в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4

Пример

Рассмотрим некоторые примеры функции now, чтобы поянть, как использовать функцию now в PostgreSQL.
Например:
