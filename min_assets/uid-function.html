UID ФУНКЦИЯ

В этом учебном пособии вы узнаете, как использоватьOracle/PLSQL функцию UID с синтаксисом и примерами.

Описание

Oracle/PLSQL функция UID возвращает целое число идентифицирующее текущего пользователя базы данных.

Синтаксис

Синтаксис Oracle/PLSQL функции UID:

UID

Параметры или аргументы

Функция UID не имеет параметров или аргументов.

Функция UID возвращает числовое значение.

Применение

Функцию UID можно использовать в следующих версияхOracle/PLSQL:
Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i, Oracle 8i

Пример

Рассмотрим пример функции Oracle UID и изучим, как использовать функцию UID вOracle/PLSQL.

Oracle PL/SQL
1
2
3
4
SELECT UID
  FROM DUAL;
 
--Результат:   55

В приведенном примере определился 55 текущий пользователь базы данных ORACLE.
