MariaDB функция REVERSE

В этом учебном пособии вы узнаете, как использовать MariaDB функцию REVERSE с синтаксисом и примерами.

Описание

MariaDB функция REVERSE возвращает строку с символами в обратном порядке.

Синтаксис

Синтаксис функции REVERSE в MariaDB:

REVERSE( string )

Параметры или аргументы

string
Исходная строка, символы которой должны быть изменены в обратном порядке.

Примечание

  • Функция REVERSE безопасна для использования с многобайтовыми данными.

Применение

Функция REVERSE может использоваться в следующих версиях MariaDB:

  • MariaDB 10

Пример
