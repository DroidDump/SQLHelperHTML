MySQL функция QUARTER

В этом учебном пособии вы узнаете, как использовать MySQL функцию QUARTER с синтаксисом и примерами.

Описание

MySQL функция QUARTER возвращает числовое значение квартала год из даты.

Синтаксис

Синтаксис MySQL функции QUARTER:

QUARTER( date_value )

Параметры или аргументы

date_value - значение даты или даты и времени, из которого извлекается квартал.

Примечание

  • Функция QUARTER возвращает четверть (число от 1 до 4) с учетом значения даты.
  • Даты, у которых месяц с Jan-Mar, вернет 1.
  • Даты, у которых есть месяц Apr-Jun, вернет 2.
  • Даты, у которых месяц Jul-Sep, вернет 3.
  • Даты, у которых месяц Oct-Dec, вернет 4.
  • См. также функции EXTRACT, YEAR, MONTH, WEEK, DAY, HOUR, MINUTE, SECOND и MICROSECOND.

Применение

Функция QUARTER может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
