COVAR_SAMP ФУНКЦИЯ

В этом учебном пособии вы узнаете, как использовать Oracle/PLSQL функцию COVAR_SAMP с синтаксисом и примерами.

Описание

Oracle/PLSQL функция COVAR_SAMP возвращает выборочную ковариацию набора пар чисел.

Синтаксис

Синтаксис Oracle/PLSQL функции COVAR_SAMP:

COVAR_SAMP( expression1, expression2 )

Параметры или аргументы

expression1, expression2 числовые выражения

Примечание

  • Функция COVAR_SAMP возвращает numeric значение.
  • Функция COVAR_SAMP удаляет все пары, где expression1 или expression2 имеет нулевое значение.

Применение

Функцию COVAR_SAMP можно использовать в следующих версиях Oracle/PLSQL:

  • Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i, Oracle 8i

Пример

Рассмотрим несколько примеров функции COVAR_SAMP и изучим, как использовать функцию COVAR_SAMP в Oracle/PLSQL.

SELECT job_id,
       COVAR_POP(SYSDATE-hire_date, salary) AS covar_pop,
       COVAR_SAMP(SYSDATE-hire_date, salary) AS covar_samp
  FROM employees
WHERE department_id in (50, 80)
GROUP BY job_id
ORDER BY job_id, covar_pop, covar_samp;
