Строковые функции MariaDB

Функция & Описание
ASCII
MariaDB функция ASCII возвращает числовое значение крайнего левого символа строки.
CHAR_LENGTH
MariaDB функция CHAR_LENGTH возвращает длину указанной строки (измеряется в символах).
CHARACTER_LENGTH
MariaDB функция CHARACTER_LENGTH возвращает длину указанной строки (измеряется в символах).
CONCAT_WS
MariaDB функция CONCAT_WS позволяет объединять два или более выражений вместе и добавляет разделитель между каждым из сцепленных выражений.
CONCAT
MariaDB функция CONCAT позволяет объединять два или более выражений вместе.
FIELD
MariaDB функция FIELD возвращает позицию значения в списке значений (val1, val2, val3, ..).
FIND_IN_SET
MariaDB функция FIND_IN_SET возвращает позицию строки в списке строк, разделенных запятыми.
FORMAT
MariaDB функция FORMAT форматирует число в формате '#,###.##', округляя его до определенного количества десятичных знаков, а затем возвращает результат в виде строки.
INSERT
MariaDB функция INSERT вставляет подстроку в строку в указанной позиции на определенное количество символов.
INSTR
MariaDB функция INSTR возвращает расположение подстроки в строке.
LCASE
MariaDB функция LCASE преобразует все символы в указанной строке в нижний регистр. Если в строке есть символы, которые не являются литералами, эта функция не влияет на них.
LEFT
MariaDB функция LEFT позволяет вам извлечь подстроку из строки, начиная с самого левого символа.
LENGTH
MariaDB функция LENGTH возвращает длину указанной строки (измеряется в байтах).
LOCATE
MariaDB функция LOCATE возвращает местоположение первого появления подстроки в строке.
LOWER
MariaDB функция LOWER преобразует все символы в указанной строке в нижний регистр. Если в строке есть символы, которые не являются буквами, то эта функция не влияет на них.
функции LPAD
MariaDB функция LPAD возвращает строку, дополненную слева частью указанной строки определенной длины.
LTRIM
MariaDB функция LTRIM удаляет все пробелы с левой стороны строки.
MID
MariaDB функция MID позволяет вам извлечь подстроку из строки.
POSITION
MariaDB функция POSITION возвращает расположение подстроки в строке.
REGEXP_INSTR
MariaDB функция REGEXP_INSTR является расширением функции INSTR. Возвращает расположение регулярного выражения pattern в string. Эта функция, представленная в MariaDB 10.0.5, позволит вам найти подстроку в строке, используя сопоставление с образцом регулярного выражения.
REGEXP_REPLACE
MariaDB функция REGEXP_REPLACE является расширением функции REPLACE. Эта функция, представленная в MariaDB 10.0.5, позволит вам заменить все вхождения подстроки в строке, используя сопоставление с образцом регулярного выражения.
REGEXP_SUBSTR
MariaDB функция REGEXP_SUBSTR является расширением функции SUBSTR. Эта функция, представленная в MariaDB 10.0.5, позволит вам извлечь подстроку в строке, используя сопоставление с образцом регулярного выражения.
REGEXP
MariaDB функция REGEXP позволяет выполнять сопоставление регулярных выражений.
REPEAT
MariaDB функция REPEAT повторяет строку указанное количество раз.
REPLACE
MariaDB функция REPLACE заменяет все вхождения указанной строки.
REVERSE
MariaDB функция REVERSE возвращает строку с символами в обратном порядке.
RIGHT
MariaDB функция RIGHT позволяет вам извлечь подстроку из строки, начиная с самого правого символа.
RPAD
MariaDB функция RPAD возвращает строку, дополненную с правой стороны указанной строки определенной длины.
RTRIM
MariaDB функция RTRIM удаляет все пробелы в правой части строки.
SPACE
MariaDB функция SPACE возвращает строку с указанным количеством пробелов.
STRCMP
MariaDB функция STRCMP проверяет, являются ли две строки одинаковыми, используя текущий набор символов.
SUBSTR
MariaDB функция SUBSTR позволяет вам извлечь подстроку из строки.
SUBSTRING_INDEX
MariaDB функция SUBSTRING_INDEX возвращает подстроку string перед number вхождений delimiter.
SUBSTRING
MariaDB функция SUBSTRING позволяет извлекать подстроку из строки.
TRIM
MariaDB функция TRIM удаляет все указанные символы в начале или в конце строки.
UCASE
MariaDB функция UCASE преобразует все символы в указанной строке в верхний регистр. Если в строке есть символы, которые не являются буквами, эта функция не влияет на них.
UPPER
MariaDB функция UPPER преобразует все символы в указанной строке в верхний регистр. Если в строке есть символы, которые не являются буквами, эта функция не влияет на них.
