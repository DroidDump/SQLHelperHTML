MariaDB условие IS NULL

В этом учебном пособии вы узнаете, как использовать условие MariaDB IS NULL с синтаксисом и примерами.

Описание

MariaDB условие IS NULL используется для проверки значения NULL в операторе SELECT, INSERT, UPDATE или DELETE.

Синтаксис

Синтаксис условия IS NULL в MariaDB:

expression IS NULL

Параметры или аргументы

expression
Значение для проверки, является ли оно значением NULL.

Примечание

  • Если expression является значением NULL, условие оценивается как TRUE.
  • Если expression не является значением NULL, условие оценивается как FALSE.
  • Смотрите также условие IS NOT NULL в MariaDB.

Пример с оператором SELECT

Рассмотрим пример использования условия IS NULL в операторе SELECT в MariaDB.
Например:

PgSQL
1
2
3
SELECT *
  FROM sites
WHERE site_name IS NULL;

Этот пример MariaDB IS NULL вернет все записи из таблицы sites, где site_name содержит значение NULL.

Пример с оператором INSERT

Далее давайте рассмотрим пример использования условия IS NULL в операторе INSERT в MariaDB.
Например:

PgSQL
1
2
3
4
5
6
INSERT INTO contacts
(contact_id, contact_name)
SELECT site_id, site_name
  FROM sites
WHERE site_name = 'Google.com'
   AND server_name IS NULL;

Этот пример вставит записи в таблицу sites, где site_name - "Google.com", а server_name содержит значение NULL.

Пример с оператором UPDATE

Далее давайте рассмотрим пример использования условия IS NULL в операторе UPDATE в MariaDB.
Например:

PgSQL
1
2
3
UPDATE sites
   SET site_name = 'Google.com'
WHERE site_name IS NULL;

В этом примере условия IS NULL будут обновлены записи в таблице sites, где site_name содержит значение NULL.

Пример с оператором DELETE

Далее давайте рассмотрим пример использования условия IS NULL в операторе DELETE в MariaDB.
Например:

PgSQL
1
2
DELETE FROM sites
WHERE server_name IS NULL;

Этот пример IS NULL удалит все записи из таблицы sites, где server_name содержит значение NULL.
