coalesce функция SQLite

В этом учебном пособии вы узнаете, как использовать SQLite функцию coalesce с синтаксисом и примерами.

Описание

Функция coalesce SQLite возвращает первое ненулевое выражение в списке.

Синтаксис

Синтаксис функции coalesce в SQLite:

coalesce( expression1, expression2, ... expression_n )

Параметры или аргументы

expression1, expression2, ... expression_n
Выражения для проверки ненулевых значений. Необходимо указать как минимум 2 выражения.

Примечание

  • Если все expressions имеют значение null, функция coalesce вернет значение null.
  • Вы должны предоставить как минимум 2 expressions в качестве параметров при использовании функции coalesce.

Применение

Функция coalesce может использоваться в следующих версиях SQLite:

  • SQLite 3.8.6, SQLite 3.8.x, SQLite 3.7.x, SQLite 3.6.x
