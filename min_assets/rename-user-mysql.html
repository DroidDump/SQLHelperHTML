Переименовать пользователя в MySQL

В этом учебном пособии вы узнаете, как использовать оператор MySQL RENAME USER с синтаксисом и примерами.

Описание

Оператор RENAME USER используется для переименования пользователя в базе данных MySQL.

Синтаксис

Синтаксис оператора RENAME USER в MySQL:

RENAME USER
user_name TO new_name;

Параметры или аргументы

user_name - имя пользователя для переименования в базе данных MySQL.
new_name - новое имя для назначения пользователю.

Пример

Рассмотрим пример, как с помощью оператора RENAME USER переименовать пользователя в MySQL.
Например:

MySQL
1
2
RENAME USER
  'samvel'@'localhost' TO 'serg'@'localhost';

В этом примере оператор RENAME USER переименовал пользователя samvel в serg в базе данных MySQL.

Переименовать более одного пользователя

Вы можете переименовать более одного пользователя за раз в MySQL. Вы можете использовать оператор RENAME USER для переименования нескольких пользователей, разделяя их запятой.
Например:

MySQL
1
2
3
RENAME USER
  'samvel'@'localhost' TO 'garigen'@'localhost',
  'serg'@'localhost' TO 'gringo'@'localhost';

Этот пример RENAME USER переименовал двух пользователей в MySQL. Первый пользователь, который назывался samvel, переименован в garigen, а второй пользователь с именем serg, переименован в gringo.
