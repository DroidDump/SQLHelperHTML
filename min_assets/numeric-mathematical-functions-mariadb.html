Числовые/математические функции MariaDB

Функция & Описание
ABS
MariaDB функция ABS возвращает абсолютное значение числа.
ACOS
MariaDB функция ACOS возвращает арккосинус числа.
ASIN
MariaDB функция ASIN возвращает арксинус числа.
ATAN
MariaDB функция ATAN возвращает арктангенс number или возвращает арктангенс n и m.
ATAN2
MariaDB функция ATAN2 возвращает арктангенс n и m.
AVG
MariaDB функция AVG возвращает среднее значение выражения.
CEIL
MariaDB функция CEIL возвращает наименьшее целочисленное значение, которое больше или равно number.
CEILING
MariaDB функция CEILING возвращает наименьшее целочисленное значение, которое больше или равно number.
COS
COS-функция MariaDB возвращает косинус числа.
COT
MariaDB функция COT возвращает котангенс number.
COUNT
MariaDB функция COUNT возвращает количество выражений.
DEGREES
MariaDB функция DEGREES преобразует значение радиана в градусы.
DIV
MariaDB функция DIV используется для целочисленного деления, где n делится на m и возвращается целочисленное значение.
EXP
MariaDB функция EXP возвращает e, возведенное в степень number (или e number).
FLOOR
MariaDB функция FLOOR возвращает наибольшее целочисленное значение, которое меньше или равно number.
GREATEST
MariaDB функция GREATEST возвращает наибольшее значение в списке выражений.
LEAST
MariaDB функция LEAST возвращает наименьшее значение в списке выражений.
LN
MariaDB функция LN возвращает натуральный логарифм числа number.
LOG
MariaDB функция LOG возвращает либо натуральный логарифм number при вызове с одним параметром, либо логарифм number с указанным основанием base, если вызывается с двумя параметрами.
LOG10
MariaDB функция LOG10 возвращает десятичный логарифм числа.
LOG2
MariaDB функция LOG2 возвращает логарифм числа с основанием 2.
MAX
MariaDB функция MAX возвращает максимальное значение выражения.
MIN
MariaDB функция MIN возвращает минимальное значение выражения.
MOD
MariaDB функция MOD возвращает остаток от n, деленного на m.
PI
MariaDB функция PI возвращает значение π (пи) отображается с 6 знаками после запятой.
POW
MariaDB функция POW возвращает значение m, возведенное в степень nth.
POWER
MariaDB функция POWER возвращает значение m, возведенное в степень nth.
RADIANS
Значение MariaDB RADIANS это значение градусов в радианах.
RAND
MariaDB функция RAND может использоваться для возврата случайного числа или случайного числа в пределах диапазона.
ROUND
MariaDB функция ROUND возвращает число, округленное до определенного количества десятичных знаков.
SIGN
MariaDB функция SIGN возвращает значение, обозначающее знак числа.
SIN
MariaDB функция SIN возвращает синус числа.
SQRT
MariaDB функция SQRT возвращает квадратный корень из number.
SUM
MariaDB функция SUM возвращает суммированное значение выражения.
TAN
MariaDB функция TAN возвращает тангенс number.
TRUNCATE
MariaDB функция TRUNCATE возвращает number, усеченное до определенного количества десятичных знаков.
