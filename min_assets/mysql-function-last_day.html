MySQL функция LAST_DAY

В этом учебном пособии вы узнаете, как использовать MySQL функцию LAST_DAY с синтаксисом и примерами.

Описание

MySQL функция LAST_DAY возвращает последний день месяца для определенной даты.

Синтаксис

Синтаксис MySQL функции LAST_DAY:

LAST_DAY( date_value )

Параметры или аргументы

date_value - дата или значение даты и времени, из которой извлекается последний день месяца.

Примечание

  • Если date_value не является допустимой датой или значением даты и времени, функция LAST_DAY вернет NULL.

Применение

Функция LAST_DAY может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1.1

    Пример
