SYSTIMESTAMP ФУНКЦИЯ

В этом учебном пособии вы узнаете, как использовать Oracle/PLSQL функцию SYSTIMESTAMP с синтаксисом и примерами.

Описание

Oracle/PLSQL функция SYSTIMESTAMP возвращает текущую системную дату и время (в том числе доли секунды и часового пояса) на ваше локальной базе данных.

Синтаксис

Синтаксис Oracle/PLSQL функции SYSTIMESTAMP:

SYSTIMESTAMP

Параметры или аргументы

Для функции SYSTIMESTAMP нет параметров или аргументов.

Функция SYSTIMESTAMP возвращает значение TIMESTAMP WITH TIME ZONE.

Применение

Функцию SYSTIMESTAMP можно использовать в следующих версиях Oracle/PLSQL:

  • Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i

Пример

Рассмотрим несколько примеров функции SYSTIMESTAMP и изучим, как использовать функцию SYSTIMESTAMP в Oracle/PLSQL.

Oracle PL/SQL
1
2
3
4
SELECT SYSTIMESTAMP
  FROM DUAL;
 
--Результат 01.08.14 09:43:50,625401 +03:00

Вы также можете использовать функцию TO_CHAR с функцией SYSTIMESTAMP.

Oracle PL/SQL
1
2
3
4
SELECT TO_CHAR(SYSTIMESTAMP, 'SSSS.FF')
  FROM DUAL;
 
--Результат 5656.083959
