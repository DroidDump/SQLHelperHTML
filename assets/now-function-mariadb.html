MariaDB функция NOW

В этом учебном пособии вы узнаете, как использовать MariaDB NOW функцию с синтаксисом и примерами.

Описание

MariaDB функция NOW возвращает текущую дату и время.

Синтаксис

Синтаксис функции NOW в MariaDB:

NOW( )

Параметры или аргументы

Для функции NOW нет параметров или аргументов.

Примечание

  • Функция NOW возвращает текущую дату в формате 'YYYY-MM-DD HH:MM:SS', если используется в строковом контексте.
  • Функция NOW возвращает текущую дату в формате YYYYMMDDHHMMSS, если она используется в числовом контексте.
  • Функции CURRENT_TIMESTAMP, LOCALTIME и LOCALTIMESTAMP являются синонимами для функции NOW .

Применение

Функция NOW может использоваться в следующих версиях MariaDB:

  • MariaDB 10
