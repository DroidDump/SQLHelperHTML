Вспомогательные функции MariaDB

Функция & Описание
BIN
MariaDB функция BIN преобразует десятичное число в двоичное число и возвращает результат в виде строкового значения.
BINARY
MariaDB функция BINARY преобразует значение в двоичную строку.
CASE
MariaDB функция CASE обладает функциональностью оператора IF-THEN-ELSE, позволяя вам оценивать условия и возвращать значение при выполнении первого условия.
CAST
MariaDB функция CAST преобразует значение из одного типа данных в другой тип данных.
COALESCE
MariaDB функция COALESCE возвращает первое NON-NULL выражение в списке.
CONNECTION_ID
MariaDB функция CONNECTION_ID возвращает идентификатор соединения для текущего соединения, который является уникальным идентификатором среди подключенных в данный момент клиентов.
CONV
MariaDB функция CONV преобразует число из одной числовой базы в другую и возвращает результат в виде строкового значения.
CONVERT
MariaDB функция CONVERT преобразует значение из одного типа данных в другой или из одного набора символов в другой.
CURRENT_USER
MariaDB функция CURRENT_USER возвращает имя пользователя и имя хоста для учетной записи MariaDB, которая использовалась сервером для аутентификации текущего клиента.
DATABASE
MariaDB функция DATABASE возвращает имя базы данных по умолчанию.
IF
MariaDB функция IF возвращает одно значение, если условие оценивается как TRUE, или другое значение, если условие оценивается как FALSE.
IFNULL
MariaDB функция IFNULL позволяет вам возвращать альтернативное значение, если expression равно NULL.
ISNULL
MariaDB функция ISNULL проверяет, является ли выражение NULL.
LAST_INSERT_ID
MariaDB функция LAST_INSERT_ID возвращает первое значение AUTO_INCREMENT, которое было установлено самой последней инструкцией INSERT или UPDATE, которая затронула столбец AUTO_INCREMENT.
NULLIF
MariaDB функция NULLIF возвращает NULL, если два выражения эквивалентны. В противном случае он возвращает первое выражение.
SESSION_USER
MariaDB функция SESSION_USER возвращает имя пользователя и имя хоста для текущего пользователя MariaDB.
SYSTEM_USER
MariaDB функция SYSTEM_USER возвращает имя пользователя и имя хоста для текущего пользователя MariaDB.
USER
MariaDB функция USER возвращает имя пользователя и имя хоста для текущего пользователя MariaDB.
VERSION
MariaDB функция VERSION возвращает версию базы данных MariaDB.
