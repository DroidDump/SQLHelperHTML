MariaDB условие IS NOT NULL

В этом учебном пособии вы узнаете, как использовать MariaDB условие IS NOT NULL с синтаксисом и примерами.

Описание

MariaDB условие IS NOT NULL используется для проверки значения NOT NULL в операторе SELECT, INSERT, UPDATE или DELETE.

Синтаксис

Синтаксис условия IS NOT NULL в MariaDB:

expression IS NOT NULL

Параметры или аргументы

expression
Значение для проверки, является ли оно ненулевым значением.

Примечание

  • Если expression НЕ является значением NULL, условие оценивается как TRUE.
  • Если expression является значением NULL, условие оценивается как FALSE.
  • Смотрите также условие IS NULL в MariaDB.

Пример с оператором SELECT

Рассмотрим пример использования условия IS NOT NULL в операторе SELECT в MariaDB.
Например:

PgSQL
1
2
3
SELECT *
  FROM sites
WHERE site_name IS NOT NULL;

Этот пример IS NOT NULL вернет все записи из таблицы sites, где site_name не содержит значения NULL.

Пример с оператором INSERT

Давайте посмотрим, как использовать условие IS NOT NULL в операторе INSERT в MariaDB.
Например:

PgSQL
1
2
3
4
5
6
INSERT INTO contacts
(contact_id, contact_name)
SELECT site_id, site_name
  FROM sites
WHERE site_name = 'Google.com'
   AND server_name IS NOT NULL;

Этот пример MariaDB IS NOT NULL вставит записи в таблицу sites, где site_name имеет значение "Google.com", а server_name не содержит значение NULL.

Пример с оператором UPDATE

Рассмотрим пример в MariaDB, как использовать условие IS NOT NULL в операторе UPDATE.
Например:

PgSQL
1
2
3
UPDATE sites
   SET server_name = 'MyServer'
WHERE site_name IS NOT NULL;

В этом примере IS NOT NULL будут обновлены записи в таблице sites, где site_name не содержит значения NULL.

Пример с оператором DELETE

Рассмотрим пример использования условия IS NOT NULL в операторе DELETE в MariaDB.
Например:

PgSQL
1
2
DELETE FROM sites
WHERE server_name IS NOT NULL;

Этот пример удалит все записи из таблицы sites, где server_name не содержит значения NULL.
