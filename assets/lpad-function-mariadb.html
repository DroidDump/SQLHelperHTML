MariaDB функции LPAD

В этом учебном пособии вы узнаете, как использовать MariaDB функцию LPAD с синтаксисом и примерами.

Описание

MariaDB функция LPAD возвращает строку, дополненную слева частью указанной строки определенной длины.

Синтаксис

Синтаксис функции LPAD в MariaDB:

LPAD( string, length, pad_string )

Параметры или аргументы

string
Строка для дополнения.
length
Длина результата после того, как string была дополнена слева.
pad_string
Указанная строка для дополнения слева string.

Примечание

  • Если длина строки превышает length, функция LPAD удалит символы из string, чтобы укоротить ее до length символов.
  • См. также функцию RPAD.

Применение

Функцию LPAD можно использовать в следующих версиях MariaDB:

  • MariaDB 10
