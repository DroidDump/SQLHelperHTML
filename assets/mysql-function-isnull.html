MySQL функция ISNULL

В этом учебном пособии вы узнаете, как использовать MySQL функцию ISNULL с синтаксисом и примерами.

Описание

MySQL функция ISNULL проверяет, является ли выражение NULL.

Синтаксис

Синтаксис MySQL функции ISNULL:

ISNULL( expression )

Параметры или аргументы

expression - значение для проверки, если NULL.

Примечание

  • Если expression равно NULL, функция ISNULL вернет 1.
  • Если expression не является значением NULL, функция ISNULL вернет 0.

Применение

Функция ISNULL может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23

Пример

Рассмотрим примеры MySQL функции ISNULL, чтобы понять, как использовать функцию ISNULL в MySQL.
