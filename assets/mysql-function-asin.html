MySQL функция ASIN

В этом учебном пособии вы узнаете, как использовать MySQL функцию ASIN с синтаксисом и примерами.

Описание

MySQL функция ASIN возвращает арксинус числа.

Синтаксис

Синтаксис MySQL функции ASIN:

ASIN( number )

Параметры или аргументы

number - число, используемое для вычисления арксинуса числа.

Примечание

  • Функция ASIN вернет NULL, если число не находится в диапазоне от -1 до 1.

Применение

Функция ASIN может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
