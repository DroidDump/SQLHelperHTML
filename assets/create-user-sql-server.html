CREATE USER SQL Server

В этом учебном пособии вы узнаете, как использовать оператор SQL Server CREATE USER с синтаксисом и примерами.

Описание

Оператор CREATE USER создает пользователя базы данных для входа в SQL Server. Пользователь базы данных сопоставляется с Login, который является идентификатором, используемым для подключения к экземпляру SQL Server.

Синтаксис

Синтаксис оператора CREATE USER в SQL Server (Transact-SQL):

CREATE USER user_name FOR LOGIN login_name;

Параметры или аргументы

user_name - имя пользователя базы данных, которого вы хотите создать.
login_name - используемый для подключения к экземпляру SQL Server. Узнайте, как создать логин.

Примечание

  • Перед созданием пользователя базы данных убедитесь, что вы сначала создали Login.
  • Вы можете отображать Login только одного пользователя в каждой базе данных на SQL Server, но вы можете сопоставить Login для разных имен пользователей для каждой базы данных, если это соотношение 1: 1 для Login в уникальную базу данных.
  • См. также оператор DROP USER.

Пример

Рассмотрим, как создать пользователя базы данных, используя оператор CREATE USER в SQL Server (Transact-SQL).
Например:

Transact-SQL
1
CREATE USER trizor FOR LOGIN trizor;

Этот пример CREATE USER создаст нового пользователя базы данных, называемого trizor в SQL Server. Для подключения к экземпляру SQL Server используется Login.
