IS NOT NULL условие SQLite

В этом учебном пособии вы узнаете, как использовать SQLite условие IS NOT NULL с синтаксисом и примерами.

Описание

SQLite условие IS NOT NULL используется для проверки значения NOT NULL в операторе SELECT, INSERT, UPDATE или DELETE.

Синтаксис

Синтаксис условия IS NOT NULL в SQLite:

expression IS NOT NULL

Параметры или аргументы

expression
Выражение для проверки, является ли оно значением NOT NULL.

Примечание

  • Если expression НЕ является значением NULL, условие оценивается как TRUE.
  • Если expression является значением NULL, условие оценивается как FALSE.

Пример - с оператором SELECT

Вот пример того, как использовать условие SQLite IS NOT NULL в инструкции SELECT:

PgSQL
1
2
3
SELECT *
FROM employees
WHERE employee_id IS NOT NULL;

Этот пример SQLite условия IS NOT NULL вернет все записи из таблицы employee, где employee_id не содержит значения NULL.

Пример - с оператором INSERT

Вот пример использования SQLite условия IS NOT NULL в операторе INSERT:

PgSQL
1
2
3
4
5
INSERT INTO temp
(temp_employee_id, temp_last_name, temp_first_name)
SELECT employee_id, last_name, first_name
FROM employees
WHERE last_name IS NOT NULL;

Этот SQLite пример IS NOT NULL вставит записи во таблицу temp, где last_name не содержит значения NULL.

Пример - с оператором UPDATE

Вот пример того, как использовать SQLite условие IS NOT NULL в инструкции UPDATE:

PgSQL
1
2
3
UPDATE employees
SET status = 'Active'
WHERE department IS NOT NULL;

Этот SQLite пример IS NOT NULL обновит записи в таблице employees и установит status на 'Active', если department не содержит значения NULL.

Пример - с оператором DELETE

Вот пример того, как использовать SQLite условие IS NOT NULL в инструкции DELETE:

PgSQL
1
2
DELETE FROM employees
WHERE termination_date IS NOT NULL;

Этот SQLite пример IS NOT NULL удалит все записи из таблицы employees, в которой значение termination_date не содержит значения NULL.
