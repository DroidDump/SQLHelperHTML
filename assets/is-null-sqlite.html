IS NULL условие SQLite

В этом учебном пособии вы узнаете, как использовать SQLite условие IS NULL с синтаксисом и примерами.

Описание

SQLite условие IS NULL используется для проверки значения NULL в операторе SELECT, INSERT, UPDATE или DELETE.

Синтаксис

Синтаксис условия IS NULL в SQLite:

expression IS NULL

Параметры или аргументы

expression
Выражение для проверки, является ли оно значением NULL.

Примечание

  • Если expression является значением NULL, условие оценивается как TRUE.
  • Если expression не является значением NULL, условие оценивается как FALSE.

Пример - с оператором SELECT

Рассмотрим пример использования SQLite условия IS NULL в операторе SELECT:

PgSQL
1
2
3
SELECT *
  FROM employees
WHERE department IS NULL;

Этот SQLite пример IS NULL вернет все записи из таблицы employees, где department содержит значение NULL.

Пример - с оператором INSERT

Далее давайте рассмотрим пример использования SQLite условия IS NULL в операторе INSERT:

PgSQL

INSERT INTO temp
(temp_employee_id, temp_last_name, temp_first_name)

       first_name
  FROM employees

Этот SQLite пример IS NULL будет вставлять записи в таблицу temp, где first_name содержит значение NULL.

Пример - с оператором UPDATE

Далее давайте рассмотрим пример использования SQLite условия IS NULL в операторе UPDATE:

PgSQL
1
2

UPDATE employees
   SET first_name = 'Unknown'
WHERE first_name IS NULL;

Этот SQLite пример IS NULL обновит записи в таблице employees и установит для first_name значение 'Unknown', где поле first_name содержит значение NULL.

Пример - с оператором DELETE

Далее давайте рассмотрим пример использования SQLite условия IS NULL в операторе DELETE:

  FROM employees
WHERE employee_id IS NULL;

Этот SQLite пример IS NULL удалит все записи из таблицы employee, где employee_id содержит значение NULL.
