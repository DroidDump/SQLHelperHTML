PostgreSQL оператор ||

В этом учебном пособии вы узнаете, как использовать оператор конкатенации PostgreSQL || с синтаксисом и примерами.

Описание

PostgreSQL оператор || позволяет объединять 2 или более строк вместе.

Синтаксис

Синтаксис для оператора || в PostgreSQL:

string1 || string2 || string_n

Параметры или аргументы

string1
Первая строка для объединения.
string2
Вторая строка для объединения.
string_n
N-ая строка для объединения.

Применение

Оператор || может быть использован в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4
