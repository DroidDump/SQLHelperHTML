MariaDB объявление переменных

В этом учебном пособии вы узнаете, как объявлять переменные в MariaDB с синтаксисом и примерами.

Что такое переменная в MariaDB?

В MariaDB переменная позволяет программисту временно хранить данные во время выполнения кода.

Синтаксис

Синтаксис объявления переменной в MariaDB:

DECLARE variable_name datatype [ DEFAULT initial_value ]

Параметры или аргументы

variable_name
Имя, которое нужно присвоить переменной.
datatype
Тип данных для назначения переменной.
DEFAULT initial_value
Необязательный. Это значение, изначально присвоенное переменной при ее объявлении. Если initial_value не указано, переменной присваивается начальное значение NULL.

Пример объявление переменной

Ниже приведен пример объявления переменной в MariaDB с именем Website.

PgSQL
1
DECLARE Website VARCHAR(45);

Затем вы можете позже установить или изменить значение переменной Website следующим образом:

PgSQL
1
SET Website = 'Bing.com';

Этот оператор SET установит для переменной Website значение 'Bing.com'.

Пример объявление переменной с начальным значением (не константа)

Ниже приведен пример того, как объявить переменную в MariaDB и дать ей начальное значение. Это отличается от константы тем, что значение переменной можно изменить позже.

PgSQL
1
DECLARE Website VARCHAR(45) DEFAULT 'Bing.com';

Это объявит переменную с именем Website как тип данных VARCHAR(45) и присвоит начальное значение 'Bing.com'.
Позже вы можете изменить значение переменной Website следующим образом:

PgSQL
1
SET Website = 'Google.com';

Этот оператор SET изменит значение переменной Website со значения 'Bing.com' на значение 'Google.com'.
