PostgreSQL функция position

В этом учебном пособии вы узнаете, как использовать в PostgreSQL функцию position с синтаксисом и примерами.

Описание

Функция position PostgreSQL возвращает местоположение подстроки в строке.

Синтаксис

Синтаксис функции position в PostgreSQL:

position( substring in string )

Параметры или аргументы

substring
Подстрока для поиска в string.
string
Строка для поиска.

Примечание

  • Первая позиция в string - 1.
  • Если substring не найден в string, то функция position вернет 0.

Применение

Функция position может использоваться в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4
