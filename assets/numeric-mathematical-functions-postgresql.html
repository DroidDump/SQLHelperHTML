Числовые/математические функции PostgreSQL

Функция & Описание
ABS
Функция abs в PostgreSQL возвращает абсолютное значение числа.
AVG
Функция avg PostgreSQL возвращает среднее значение выражения.
CEIL
Функция ceil PostgreSQL возвращает наименьшее целочисленное значение, которое больше или равно number.
CEILING
Функция ceiling PostgreSQL возвращает наименьшее целочисленное значение, которое больше или равно number.
COUNT
Функция count PostgreSQL возвращает количество выражений.
DIV
Функция div PostgreSQL используется для целочисленного деления, где n делится на m и возвращается целочисленное значение.
EXP
Функция exp в PostgreSQL возвращает e, возведенное в степень number (или enumber).
FLOOR
Функция floor PostgreSQL возвращает наибольшее целочисленное значение, которое меньше или равно number.
MAX
Функция max PostgreSQL возвращает максимальное значение выражения.
MIN
Функция min в PostgreSQL возвращает минимальное значение выражения.
MOD
Функция mod PostgreSQL возвращает остаток от n, разделенный на m.
POWER
Функция power PostgreSQL возвращает значение m, возведенное n-ную в степень.
RANDOM
Функция random PostgreSQL может использоваться для возврата случайного числа или случайного числа в пределах диапазона.
ROUND
Функция round PostgreSQL возвращает число, округленное до определенного количества десятичных знаков.
SETSEED
Функция setseed PostgreSQL может использоваться для установки начального числа при следующем вызове функции random. Если вы не вызываете setseed, PostgreSQL будет использовать свое собственное значение seed. Это может или не может быть действительно random.
Если вы установили начальное значение, вызвав функцию setseed, тогда функция random вернет повторяемую последовательность случайных чисел, полученную из начального числа.
Подсказка: Функция setseed может быть очень полезна в ситуациях, когда вы хотите убедиться, что PostgreSQL использует действительно случайное начальное число.
SIGN
Функция sign PostgreSQL возвращает значение, обозначающее знак числа.
SQRT
Функция sqrt PostgreSQL возвращает квадратный корень из number.
SUM
Функция sum PostgreSQL возвращает суммированное значение выражения.
TRUNC
Функция trunc PostgreSQL возвращает number, усеченное до определенного количества десятичных знаков.
