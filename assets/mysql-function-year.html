MySQL функция YEAR

В этом учебном пособии в узнаете, как использовать MySQL функцию YEAR с синтаксисом и примерами.

Описание

MySQL функция YEAR возвращает год из значения даты.

Синтаксис

Синтаксис MySQL функции YEAR:

YEAR( date_value )

Параметры или аргументы

date_value - значение даты или даты/времени, из которого извлекается год.

Примечание

  • Функция YEAR возвращает четырехзначный год (число от 1000 до 9999), учитывая значение даты.
  • См. также функции EXTRACT, QUARTER, MONTH, DAY, HOUR, MINUTE, SECOND и MICROSECOND.

Применение

Функция YEAR может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23

Пример
