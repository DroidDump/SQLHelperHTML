MariaDB функция ASCII

В этом учебном пособии вы узнаете, как использовать MariaDB функцию ASCII с синтаксисом и примерами.

Описание

MariaDB функция ASCII возвращает числовое значение крайнего левого символа строки.

Синтаксис

Синтаксис функции ASCII в MariaDB:

ASCII( single_character )

Параметры или аргументы

single_character
Указанный символ, для которого требуется получить код NUMBER. Если введено более одного символа, функция ASCII вернет значение для первого символа и проигнорирует все символы после первого.

Применение

Функцию ASCII можно использовать в следующих версиях MariaDB:

  • MariaDB 10
