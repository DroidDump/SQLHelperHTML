ltrim функция SQLite

В этом учебном пособии вы узнаете, как использовать SQLite функцию ltrim с синтаксисом и примерами.

Описание

SQLite функция ltrim удаляет все указанные символы из левой части строки.

Синтаксис

Синтаксис функции ltrim в SQLite:

ltrim( string, [ trim_string ] )

Параметры или аргументы

string
Строка для обрезки символов с левой стороны.

trim_string
Строка, которая будет удалена с левой стороны string. Если этот параметр пропущен, функция ltrim удалит все начальные пробелы из string.

Примечание

  • Смотрите также функции rtrim и trim.

Применение

Функция ltrim может использоваться в следующих версиях SQLite:

  • SQLite 3.8.6, SQLite 3.8.x, SQLite 3.7.x, SQLite 3.6.x
