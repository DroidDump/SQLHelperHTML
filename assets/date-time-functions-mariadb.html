Дата / время функции MariaDB

Функция & Описание
ADDDATE
MariaDB функция ADDDATE возвращает дату, после которой был добавлен определенный интервал даты/времени.
ADDTIME
MariaDB функция ADDTIME возвращает значение времени / даты и времени, после которого был добавлен определенный интервал времени.
CURDATE
MariaDB функция CURDATE возвращает текущую дату.
CURRENT_DATE
MariaDB функция CURRENT_DATE возвращает текущую дату.
CURRENT_TIME
MariaDB функция CURRENT_TIME возвращает текущее время.
CURRENT_TIMESTAMP
MariaDB функция CURRENT_TIMESTAMP возвращает текущую дату и время.
CURTIME
MariaDB функция CURTIME возвращает текущее время.
DATE_ADD
MariaDB функция DATE_ADD возвращает дату, после которой был добавлен определенный интервал времени/даты.
DATE_FORMAT
MariaDB функция DATE_FORMAT форматирует дату в соответствии с маской формата.
DATE_SUB
MariaDB функция DATE_SUB возвращает дату, после которой вычитается определенный интервал даты/времени.
DATE
MariaDB функция DATE извлекает значение даты из выражения даты или даты и времени.
DATEDIFF
MariaDB функция DATEDIFF возвращает разницу в днях между двумя значениями даты.
DAY
MariaDB функция DAY возвращает дневную часть значения даты.
DAYNAME
MariaDB функция DAYNAME возвращает название дня недели для даты.
DAYOFMONTH
MariaDB функция DAYOFMONTH возвращает дневную часть значения даты.
DAYOFWEEK
MariaDB функция DAYOFWEEK возвращает индекс дня недели для значения даты.
DAYOFYEAR
MariaDB функция DAYOFYEAR возвращает день года для значения даты.
EXTRACT
MariaDB функция EXTRACT извлекает детали из даты.
FROM_DAYS
MariaDB функция FROM_DAYS преобразует числовой день в значение даты.
HOUR
MariaDB функция HOUR возвращает часовую часть значения даты.
LAST_DAY
MariaDB функция LAST_DAY возвращает последний день месяца для данной даты.
LOCALTIME
MariaDB функция LOCALTIME возвращает текущую дату и время.
LOCALTIMESTAMP
MariaDB функция LOCALTIMESTAMP возвращает текущую дату и время.
MAKEDATE
MariaDB функция MAKEDATE возвращает дату для определенного года и значение дня в году.
MAKETIME
MariaDB функция MAKETIME возвращает время для определенной комбинации часов, минут и секунд.
MICROSECOND
MariaDB функция MICROSECOND возвращает микросекундную часть значения даты.
MINUTE
MariaDB функция MINUTE возвращает минутную часть значения даты.
MONTH
MariaDB функция MONTH возвращает месячную часть значения даты.
MONTHNAME
MariaDB функция MONTHNAME возвращает полное название месяца для даты.
NOW
MariaDB функция NOW возвращает текущую дату и время.
PERIOD_ADD
MariaDB функция PERIOD_ADD берет период (в формате YYMM или YYYYMM) и добавляет к нему указанное количество месяцев.
PERIOD_DIFF
MariaDB функция PERIOD_DIFF возвращает разницу в месяцах между двумя периодами (в формате YYMM или YYYYMM).
QUARTER
MariaDB функция QUARTER возвращает квартальную часть значения даты.
SEC_TO_TIME
MariaDB функция SEC_TO_TIME преобразует числовые секунды в значение времени.
SECOND
MariaDB функция SECOND возвращает секундную часть значения даты.
STR_TO_DATE
MariaDB функция STR_TO_DATE принимает строку и возвращает дату, указанную маской формата.
SUBDATE
MariaDB функция SUBDATE возвращает дату, после которой вычитается определенный интервал даты/времени.
SUBTIME
MariaDB функция SUBTIME возвращает значение даты/времени или времени, после которого вычитается определенный интервал времени.
SYSDATE
MariaDB функция SYSDATE возвращает текущую дату и время.
TIME_FORMAT
MariaDB функция TIME_FORMAT форматирует время в соответствии с маской формата.
TIME_TO_SEC
MariaDB функция TIME_TO_SEC преобразует значение времени в цифровые секунды.
TIME
MariaDB функция TIME извлекает значение времени из выражения время/дата или время.
TIMEDIFF
MariaDB функция TIMEDIFF возвращает разницу (выраженную в виде значения времени) между двумя значениями даты/времени / времени.
TIMESTAMP
MariaDB функция TIMESTAMP преобразует expression в значение даты и времени и, если указано, добавляет к значению необязательный интервал времени interval.
TO_DAYS
MariaDB функция TO_DAYS преобразует дату в числовые дни.
WEEK
MariaDB функция WEEK возвращает недельную часть значения даты.
WEEKDAY
MariaDB функция WEEKDAY возвращает индекс дня недели для даты.
WEEKOFYEAR
MariaDB функция WEEKOFYEAR возвращает неделю года для значения даты.
YEAR
MariaDB функция YEAR возвращает годовую часть из значения даты.
YEARWEEK
MariaDB функция YEARWEEK возвращает год и неделю для значения даты.
