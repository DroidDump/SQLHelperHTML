MySQL функция LOG

В этом учебном пособии вы узнаете, как использовать MySQL функцию LOG с синтаксисом и примерами.

Описание

MySQL функция LOG возвращает либо натуральный логарифм числа, если вызывается с одним параметром, либо логарифм числа по основанию, если вызывается с двумя параметрами.

Синтаксис

Синтаксис MySQL функции LOG:

LOG( number )

ИЛИ

LOG( base, number )

Параметры или аргументы

number - число, для которого берется натуральный логарифм. Должно быть больше 0.

base - Основание, на которое рассчитывается натуральный логарифм. Должно быть больше 1.

Примечание

  • Функция LOG возвращает NULL, если number меньше или равно 0.
  • Функция LOG возвращает NULL, если base меньше или равна 1.
  • См. также функции LN и EXP.

Применение

Функция LOG может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
