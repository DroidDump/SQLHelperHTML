SQL Server функция LOWER

В этом учебном пособии вы узнаете, как использовать функцию LOWER в SQL Server (Transact-SQL) с синтаксисом и примерами.

Описание

В SQL Server (Transact-SQL) функция LOWER преобразует все буквы в указанной строке в нижний регистр. Если в строке есть символы, которые не являются буквами, они не зависят от этой функции.

Синтаксис

Синтаксис функции LOWER в SQL Server (Transact-SQL):

LOWER( string )

Параметры или аргументы

string - строка для преобразования в нижний регистр.

Примечание

  • См. Также функцию UPPER.

Применение

Функция LOWER может использоваться в следующих версиях SQL Server (Transact-SQL):
SQL Server vNext, SQL Server 2016, SQL Server 2015, SQL Server 2014, SQL Server 2012, SQL Server 2008 R2, SQL Server 2008, SQL Server 2005
