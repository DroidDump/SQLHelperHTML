PostgreSQL функция character_length

В этом учебном пособии вы узнаете, как использовать в PostgreSQL функцию character_length с синтаксисом и примерами.

Описание

Функция character_length в PostgreSQL возвращает длину указанной строки.

Синтаксис

Синтаксис функции character_length в PostgreSQL:

character_length( string )

Параметры или аргументы

string
Строка, для которой возвращается длина.

Примечание

  • См. Также функции char_length и length.

Применение

Функция character_length может использоваться в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4
