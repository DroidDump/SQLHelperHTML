upper функция SQLite

В этом учебном пособии вы узнаете, как использовать SQLite функцию upper с синтаксисом и примерами.

Описание

SQLite функция upper преобразует все символы в указанной строке в верхний регистр. Если в строке есть символы, которые не являются буквами, эта функция их не затрагивает.

Синтаксис

Синтаксис функции upper в SQLite:

upper( string )

Параметры или аргументы

string
Строка для преобразования в верхний регистр.

Примечание

  • Функция upper преобразует в верхний регистр только символы ASCII.

Применение

Функция upper может использоваться в следующих версиях SQLite:

  • SQLite 3.8.6, SQLite 3.8.x, SQLite 3.7.x, SQLite 3.6.x
