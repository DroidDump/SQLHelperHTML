Функции шифрования MariaDB

Функция & Описание
ENCRYPT
MariaDB функция ENCRYPT используется для шифрования строки с помощью UNIX crypt ().
MD5
MariaDB функция MD5 возвращает 128-битное представление контрольной суммы MD5 для строки.
OLD_PASSWORD
MariaDB функция OLD_PASSWORD используется системой аутентификации для генерации хешированного пароля из строки открытого текста, используя более старые методы хеширования. Чтобы использовать более новые методы хеширования, используйте функцию PASSWORD.
PASSWORD
MariaDB функция PASSWORD используется системой аутентификации в MariaDB для генерации хешированного пароля из строки открытого текста с использованием более мощных методов хеширования. Чтобы использовать более старые методы хеширования, используйте OLD_PASSWORD функцию.
